# Codex Quota 2.3.0

Codex Quota 2.3.0 introduces four real WidgetKit visual themes without changing the sanitized snapshot, network, or privacy boundaries. Existing widgets remain installed and now follow the app-wide theme; the new configurable widget can choose a different theme per instance.

Codex Quota 2.3.0 新增四套真实 WidgetKit 视觉主题，同时保持脱敏快照、网络请求与隐私边界不变。升级后已有小组件不会消失，并自动跟随应用全局主题；新增的可配置小组件支持每个实例独立选主题。

## Install / 安装

### DMG

Download `Codex-Quota-macOS-universal.dmg`, open it, and drag **Codex Quota** into **Applications**.

下载 `Codex-Quota-macOS-universal.dmg`，打开后将 **Codex Quota** 拖入 **Applications（应用程序）**。

### One-command installer / 一行安装

```bash
curl -fsSL https://raw.githubusercontent.com/Zamisku/Codex-Quota/main/scripts/install-release.sh | /bin/zsh
```

The installer verifies the published SHA-256, code signature, host and widget Bundle IDs, backs up the current installation, registers the widget, and launches the app.

安装器会核验公开的 SHA-256、代码签名、宿主与 Widget Bundle ID，备份当前安装、注册小组件并启动应用。

## What changed / 更新内容

- Added Crystal Glass, Full-card Aquarium, Dual-track Orbit, and Minimal Aurora themes for both Small and Medium widgets.
- Added a 2 × 2 in-app gallery with real Small/Medium previews and motion demonstrations.
- Preserved the original **Codex Quota** widget kind as the globally themed, upgrade-safe option.
- Added **Codex Quota · Custom**, with “Follow App” and four per-instance theme choices.
- Added one-shot update animations, macOS 26 Liquid Glass controls, macOS 14/15 material fallback, and adaptations for reduced motion, reduced transparency, monochrome rendering, and VoiceOver.
- Refreshed the English and Chinese project introductions, widget screenshots, and promotional artwork.

- Small 与 Medium 均新增“晶透玻璃、整卡水族箱、双轨星环、极光简约”四套主题。
- 应用内新增 2 × 2 主题画廊，可切换真实 Small/Medium 预览并展示完整动效。
- 原有 **Codex Quota** Widget kind 保持不变，升级后继续存在并跟随全局主题。
- 新增 **Codex Quota · 自定义**，支持“跟随应用”及四套单实例主题。
- 新增一次性刷新动效、macOS 26 Liquid Glass 控件、macOS 14/15 材质降级，以及减少动态、减少透明度、单色渲染和 VoiceOver 适配。
- 中英文项目介绍、Widget 截图和宣传素材已全面更新。

## First-launch notice / 首次启动说明

This build is Apple Development-signed but not notarized. When installing through Finder, Control-click the app, choose **Open**, and confirm once. The command-line installer downloads and verifies the same release without requiring Xcode.

当前构建使用 Apple Development 签名，但尚未完成 Apple 公证。通过 Finder 安装时，请按住 Control 点击应用，选择“打开”并确认一次。一行安装器下载并校验同一发行包，全程不需要 Xcode。

Verify manual downloads against `SHA256SUMS` before installation.

手动下载时，请在安装前使用 `SHA256SUMS` 核验文件。
