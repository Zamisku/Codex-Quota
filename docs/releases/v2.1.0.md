# Codex Quota 2.1.0

Codex Quota is now available as a prebuilt Universal macOS app. Ordinary users no longer need Xcode, XcodeGen, source code, or an Apple Developer account.

## Install / 安装

### DMG

Download `Codex-Quota-macOS-universal.dmg`, open it, and drag **Codex Quota** into **Applications**.

下载 `Codex-Quota-macOS-universal.dmg`，打开后将 **Codex Quota** 拖入 **Applications（应用程序）**。

### One-command installer / 一行安装

```bash
curl -fsSL https://raw.githubusercontent.com/Zamisku/Codex-Quota/main/scripts/install-release.sh | /bin/zsh
```

The installer verifies the published SHA-256 checksum, code signature, application and widget Bundle IDs, backs up an existing installation, registers the widget, and launches the app.

安装脚本会核验公开的 SHA-256、代码签名、应用及小组件 Bundle ID，备份旧版本、注册小组件并启动应用。

## First-launch notice / 首次启动说明

This build is signed with Apple Development but is not yet notarized. When installing through Finder, Control-click the app, choose **Open**, and confirm once. The command-line installer downloads and verifies the same release without requiring Xcode. Future production releases can become fully notarized after Developer ID secrets are configured in the included Release workflow.

当前构建使用 Apple Development 签名，但尚未完成 Apple 公证。通过 Finder 安装时，请按住 Control 点击应用，选择“打开”并确认一次。一行安装器下载并校验同一发行包，全程不需要 Xcode。仓库已包含 Developer ID 与公证工作流，配置发行密钥后即可发布完整公证版本。
