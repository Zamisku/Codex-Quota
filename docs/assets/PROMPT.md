# Promotional background generation brief

The final `promo-background.png` was produced with the built-in OpenAI ImageGen workflow. The app icon, `widget-small.png`, and `widget-medium.png` were supplied as visual-language references only.

```text
Use case: ads-marketing
Asset type: reusable premium background for a GitHub README hero and repository social preview.
Reference-image roles: the supplied Codex Quota app icon and WidgetKit previews are visual-language references only. Borrow their airy sky-blue, aqua, mint, white, and deep navy palette, their calm native-macOS feeling, and their soft rounded sensibility. Do not copy, redraw, reinterpret, or include any logo, letter, number, UI card, widget, screenshot, gauge, icon, interface element, or readable content from the references.
Primary request: create a polished abstract macOS-inspired brand backdrop for an open-source native quota-monitoring utility.
Scene/background: luminous sky-blue to pale aqua to fresh mint atmosphere, subtle layered translucent glass ribbons, very soft aurora glow, restrained depth, faint fine-grain texture, and a few elegant diffused light pools. Premium Apple-platform editorial finish without imitating any specific Apple artwork.
Composition: extra-wide 2:1 landscape composition. Keep the left 42% calm, bright, and low-detail for a dark headline overlay. Keep the right half visually richer but still uncluttered for later placement of real product screenshots. Preserve generous safe margins on all sides. Avoid a central focal object.
Lighting/mood: optimistic, trustworthy, precise, privacy-conscious, quiet confidence, soft studio illumination.
Palette: #78B7F2 sky blue, #C8EAF2 pale aqua, #A8EBC8 mint, #F4FAFF near-white, subtle #15283F navy only as very faint depth accents.
Constraints: background artwork only; absolutely no text, typography, symbols, logos, letters, numbers, watermarks, UI, widgets, devices, browser frames, app windows, mockup frames, or hard-edged product cards. No busy particles, no neon cyberpunk look, no excessive bloom. Must remain readable under later deterministic dark-navy text and exact screenshot overlays.
```
