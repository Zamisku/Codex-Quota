# Promotional background generation brief

The final `promo-background.png` was produced with the built-in OpenAI ImageGen workflow. The app icon, `widget-small.png`, and `widget-medium.png` were supplied as visual-language references only.

```text
Use case: ads-marketing
Asset type: reusable premium background for a GitHub README hero and repository social preview.
Reference-image roles: the supplied Codex Quota app icon and WidgetKit previews are visual-language references only. Borrow their airy sky-blue, aqua, mint, white, and deep navy palette, their calm native-macOS feeling, and their soft rounded sensibility. Do not copy, redraw, reinterpret, or include any logo, letter, number, UI card, widget, screenshot, gauge, icon, interface element, or readable content from the references.
Primary request: create a polished abstract macOS-inspired brand backdrop for an open-source native quota-monitoring utility.
Scene/background: luminous sky-blue to pale aqua to fresh mint atmosphere, subtle layered translucent glass ribbons, very soft aurora glow, restrained depth, faint fine-grain texture, and a few elegant diffused light pools. Premium Apple-platform editorial finish without imitating any specific Apple artwork.
Composition: extra-wide 2:1 landscape composition. Keep the left 42% calm, bright, and low-detail for a dark headline overlay. Keep the right half visually richer but still uncluttered for later placement of real product screenshots. Preserve generous safe margins on all sides. Avoid a central focal object.
Lighting/mood: optimistic, trustworthy, precise, privacy-conscious, quiet confidence, soft studio illumination.
Palette: #78B7F2 sky blue, #C8EAF2 pale aqua, #A8EBC8 mint, #F4FAFF near-white, subtle #15283F navy only as very faint depth accents.
Constraints: background artwork only; absolutely no text, typography, symbols, logos, letters, numbers, watermarks, UI, widgets, devices, browser frames, app windows, mockup frames, or hard-edged product cards. No busy particles, no neon cyberpunk look, no excessive bloom. Must remain readable under later deterministic dark-navy text and exact screenshot overlays.
```

## Four-theme gallery background

The final `theme-showcase-background-v1.png` was generated from the following text-only brief. No UI screenshot or logo was supplied as a target image.

```text
Use case: ads-marketing.
Asset type: reusable premium abstract background for a GitHub README product showcase.
Primary request: create one polished 16:10 landscape backdrop that visually unifies four distinct quota-widget themes without depicting any interface.
Visual structure: four softly connected atmospheric territories with no hard grid lines. Upper-left: luminous crystal glass, pale sky blue, subtle refraction caustics and a precise circular lens halo. Upper-right: clear cyan aquarium atmosphere with one elegant horizontal water-surface ripple and a few tiny diffused bubbles. Lower-left: deep navy space with restrained concentric orbital arcs, faint star dust, and cyan-indigo rim light. Lower-right: airy blue-to-mint aurora veils and soft bloom. Blend the territories through a calm central pearlescent transition so the whole image feels like one coherent premium native desktop product.
Composition: 16:10 landscape. Leave generous low-detail safe zones through the center of each quadrant for later placement of four real product cards. Keep all decorative detail near outer edges and behind where cards will sit. Strong but tasteful depth, clean editorial finish, subtle fine grain.
Mood: precise, calm, trustworthy, modern, privacy-conscious, delightful but not playful.
Palette: pale ice #EAF7FF, sky #78B7F2, cyan #28C7E8, mint #9BE8C3, navy #0B1733, indigo #353B91. Use coral or amber only as extremely small reflected accents.
Constraints: background artwork only. Absolutely no text, letters, numbers, logos, icons, gauges, UI cards, widgets, app windows, devices, browser frames, borders, mockups, watermarks, or recognizable interface elements. No central focal object, no busy particles, no cyberpunk neon, no excessive bloom.
```

## Four-theme social-preview background

The final `theme-social-background-v1.png` was generated from the following text-only brief. Exact typography, iconography, and all four Widget previews are composited afterward by `scripts/render-theme-assets.swift`.

```text
Use case: ads-marketing.
Asset type: premium 2:1 GitHub repository social-preview background.
Primary request: create a bold, highly legible abstract landscape backdrop for a native macOS quota-monitoring app with four visual themes, intended for later deterministic overlay of exact typography, an app icon, and real widget screenshots.
Composition: exact wide 2:1 feeling. Keep the left 38% calm, dark navy-to-indigo, low-detail, with ample negative space for a large white product name and short tagline. Let the right 62% carry a graceful upward sweep combining transparent crystal refraction, a clean cyan water ribbon with sparse bubbles, two restrained orbital arcs, and a mint-blue aurora glow. The four motifs should read as one elegant stream rather than four separate boxes. Keep the center-right sufficiently calm for later product cards. Strong thumbnail readability and generous safe margins.
Lighting: premium studio light, crisp edge highlights, controlled depth, subtle fine grain, quiet confidence.
Palette: deep navy #07152E, cobalt #2456B8, luminous cyan #28C7E8, pale ice #EAF7FF, mint #9BE8C3, indigo #403FA0. Tiny restrained amber/coral reflected glints are acceptable.
Constraints: background artwork only. Absolutely no text, letters, numbers, logos, icons, UI, widgets, cards, app windows, devices, browser frames, gauges, typography, mockups, borders, or watermarks. No cyberpunk aesthetic, no busy star field, no excessive bloom, no central emblem.
```
